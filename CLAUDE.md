# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a sandbox/testing directory used to explore Claude Code capabilities. No build system, test runner, or framework is configured.

## Permissions

The following tool permissions are pre-configured in `.claude/settings.local.json`:

- `WebFetch` from `raw.githubusercontent.com`
- `Bash(npm install:*)` — npm installs are auto-allowed
